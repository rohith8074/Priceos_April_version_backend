4.3 — The 4-Pass Waterfall Engine
PassNameWhat Runs
Pass 1FoundationADMIN_BLOCK + EVENT + SEASON rules applied by date range and priority. Sets base price from rules, min stay, CICO, block flags. Highest priority rule wins.
Pass 2StrategyLast-minute discount (flat or ramp curve) · Far-out premium · DOW adjustment · Occupancy-based adjustment (high/low threshold). Applied in sequence to price.
Pass 3InventoryGap detection from InventoryMaster booking map. Gap prevention blocks short orphans. Gap fill discounts and CICO override for fill-eligible gaps.
Pass 4Guardrails / IntegrityWeekend minimum price check → then priceFloor clamp → then priceCeiling clamp → lowestMinStayAllowed clamp. Round to 2 decimal places.

6.1 — What Is an Insight?

Insights are AI-generated revenue recommendations surfaced from signal detectors. They are NOT price proposals — they are advisory signals that tell the operator why something needs attention and what to do about it. The operator can approve, modify, snooze, or dismiss each one.

Insight Data Fields
category · severity (high/medium/low) · status (pending/approved/modified/rejected/snoozed/superseded)
title · summary · confidence (0–100%) · action (type, adjustPct, absolutePrice, dateRange, scope)
listingId · createdAt · snoozeUntil
12 Insight Categories
